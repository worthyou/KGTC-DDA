# KGTC-DDA
Drug-disease association prediction based on Parallel Graph Isomorphism and Transformer Networks with Cross-Attention

# Environment Requirments:
- pytorch 2.0.1
- python 3.9.21
- cudatoolkit 11.7.0
- numpy 1.23.5
- scikit-learn 1.6.1

# Data:
The data files needed to run the model, which contain F-dataset, C-dataset and lrssl.

# Usage:
Execute ```python main.py```
